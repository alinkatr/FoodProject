# Техническое задание на проект: FreshTracker

## 1. Название и концепция
**Название:** FreshTracker  
**Концепция:** Веб-сервис для домашнего учёта сроков годности продуктов питания с персонализированными рекомендациями по их использованию. Решает проблему пищевых отходов с помощью аналитики и рекомендательных алгоритмов.

## 2. Целевая аудитория и роли
- **Гость:** Просмотр демо-статистики, регистрация, вход
- **Пользователь:** Полный доступ к личному кабинету, продуктам, рекомендациям
- **Администратор:** Управление категориями, шаблонами рекомендаций, пользователями

## 3. Схема данных (3 кастомные модели)

### Модель 1: Category
- `name` - название категории
- `default_shelf_life_days` - рекомендуемый срок хранения
- `icon` - иконка категории

### Модель 2: Product
- `user` (ForeignKey) - владелец
- `name` - название продукта
- `category` (ForeignKey) - категория
- `expiration_date` - срок годности
- `purchase_date` - дата покупки
- `quantity`, `unit` - количество и единица
- `storage`, `priority` - место хранения и приоритет
- `status` - статус (активный/использован/просрочен)
- `notes` - заметки
- `notifications` - уведомления

### Модель 3: RecommendationTemplate
- `category` (ForeignKey) - для какой категории
- `days_before_expiry` - за сколько дней показывать
- `title`, `text` - заголовок и текст
- `action_text` - текст действия
- `icon` - иконка
- `is_active` - активно

## 4. Основные сценарии пользователя
1. **Регистрация и авторизация:** Пользователь регистрируется и входит в систему
2. **Добавление продукта:** Пользователь через форму добавляет продукт с указанием срока годности
3. **Просмотр и фильтрация:** Пользователь видит список продуктов с фильтрацией по категориям и статусам
4. **Получение рекомендаций:** Система анализирует сроки и предлагает рекомендации
5. **Аналитика:** Пользователь видит графики распределения продуктов и статистику
6. **Отметка использованного:** Пользователь отмечает использованные продукты

## 5. Технический стек и интеграции
- **Django 4.0** - основной фреймворк
- **Pandas + Matplotlib** - анализ данных и визуализация
- **Bootstrap 5** - фронтенд
- **SQLite** - база данных (для разработки)
