# FreshTracker - Система учёта сроков годности продуктов

![Django](https://img.shields.io/badge/Django-4.0-green)
![Python](https://img.shields.io/badge/Python-3.9-blue)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5.0-purple)

##  О проекте

FreshTracker - это веб-приложение для домашнего учёта сроков годности продуктов питания с персонализированными рекомендациями по их использованию. Сервис помогает сократить пищевые отходы и экономить бюджет.

##  Функциональность

### Для пользователей:
-  Учет сроков годности продуктов
-  Персональные рекомендации по использованию
-  Визуальная аналитика с графиками
-  Фильтрация и сортировка продуктов
-  Авторизация пользователей
-  Отслеживание статуса продуктов

### Для администраторов:
-  Панель управления категориями
-  Управление шаблонами рекомендаций
-  Просмотр статистики пользователей
-  Управление продуктами

##  Технологический стек

### Backend
- **Django 4.0** - веб-фреймворк
- **SQLite** - база данных (для разработки)
- **Pandas** - анализ данных
- **Matplotlib** - визуализация данных

### Frontend
- **Bootstrap 5** - адаптивный дизайн
- **Font Awesome** - иконки
- **HTML5/CSS3** - верстка
- **JavaScript** - клиентская логика

##  Data Science компоненты

Проект использует:
- **Pandas** для анализа статистики продуктов
- **Matplotlib** для генерации графиков
- Алгоритмы рекомендаций на основе анализа сроков годности

##  Архитектура

### Модели данных
1. **Category** - категории продуктов (молочные, овощи и т.д.)
2. **Product** - продукты пользователей с полной информацией
3. **RecommendationTemplate** - шаблоны рекомендаций для категорий

### Основные функции
- Главная страница с аналитикой
- Список продуктов с фильтрацией
- Добавление/редактирование продуктов
- Рекомендательная система
- Статистика с графиками
